# ビルドと動作確認の手順

プロジェクト設定の修正が完了しました。以下の手順でクリーンビルドと動作確認を行ってください。

## 1. Gowin EDAでのクリーンビルド

「Nothing is executed」と表示されるのを防ぐため、以下の手順でビルド状態をリセットします。

1.  **Gowin EDAを再起動**（念のため `VGA_BW_Tang25K.gprj` を開き直してください）。
2.  左側の **Design** タブで `VGA_BW_first_Tang25K.v` が**太字**（Top Module）になっているか確認します。なっていなければ右クリックして `Set as Top Module` を選択してください。
3.  メニューの **Process** -> **Clean All** を実行します。
4.  次に **Process** -> **Rebuild All** を実行します。

## 2. ビットストリームの書き込み

ビルドが成功したら、生成されたファイルを書き込みます。

1.  **Programmer** を開きます。
2.  以下のパスのファイルが選択されているか確認します：
    `c:\Users\asami\Documents\verilog\impl\pnr\VGA_BW_Tang25K.fs`
3.  **Program/Configure** を実行し、**"finished"** と表示されることを確認します。

## 3. オシロスコープでの検証

信号が正しく出力されているか、以下のポイントを測定してください。

| 信号 | 測定ピン | 期待されるタイミング |
| :--- | :--- | :--- |
| **HSYNC** | J5 - E11 | 周期: 約31.7μs (約31.5kHz) |
| **VSYNC** | J5 - D11 | 周期: 約16.7ms (約60Hz) |
| **VIDEO** | J5 - C11/B11/A11 | 有効表示期間中にパルス（白黒の模様） |

> [!IMPORTANT]
> 信号が全く出ていない（0.76Vのままなど）場合は、FPGAの電源を一度切り、JTAGケーブル（USB）を挿し直してから再度プログラミングを試みてください。

## 4. モニターへの接続

信号が正常であれば、VGAケーブルを接続して市松模様が表示されるか確認してください。
